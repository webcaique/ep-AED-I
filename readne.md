# EP DE ALGORÍTMO E ESTRUTURAS DE DADOS I
## Sobre
Este projeto tem o objetivo de implementar dois tipos de estruturas de dados: Lista e Árvore, e comparar a performace de ambas as estruturas.
O programa lê um arquivo de texto e armazena cada palavra na estrutura escolhida pelo usuário quando o programa é iniciado.
Durante a execução, o usuário pode busca palavras no texto, em que pode imprimir na tela em quais linha do texto está a palavra, suas ocorrências e quantas comparações realizadas ou a palavra não existe, retornando a quantidade de comparações.
O usuário pode iniciar a busca digitando *busca palavra* ou pode finalizar o programa digitando *fim*.

### Estruturas usadas
Lista Sequência e Árvore AVL.
